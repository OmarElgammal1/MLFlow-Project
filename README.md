# Bank Consumer Churn Prediction

This project predicts bank consumer churn using a logistic regression model. The experiment is tracked with MLflow.

## Dataset

The dataset used is `Churn_Modelling.csv`, which can be found in the `dataset` directory.

## How to run

1. Install the dependencies:
```bash
pip install -r requirements.txt
```

2. Run the training script:
```bash
python src/train.py
```

## Dependencies

The dependencies for this project are listed in the `requirements.txt` file.
